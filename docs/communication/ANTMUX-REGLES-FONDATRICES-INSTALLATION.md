## Règles normatives enregistrées

1. **Disque dédié** — Le projet dispose de son propre disque dur.
2. **Nom du disque** — Le disque dur porte le nom **Antmux**.
3. **Installation à la racine** — L’environnement principal issu de Codex CLI doit être installé directement à la racine du disque Antmux.
4. **Identité du système** — Dans le projet, le système ne doit pas être présenté ni nommé « Codex »; son identité opérationnelle est **Antmux**.
5. **Indépendance totale** — Toute installation, dépendance, configuration, cache, donnée, journal, outil, environnement ou composant appartenant au projet doit demeurer sur le disque Antmux.
6. **Interdiction hors disque** — Aucun élément du projet ne doit être installé volontairement sur le disque système, dans le profil utilisateur ou ailleurs hors du disque Antmux.
7. **Première travailleuse** — La première travailleuse du système est **Linuxia**, désignée comme **la reine Linuxia**.

## Principe absolu

> **Tout ce qui appartient à Antmux reste sur le disque Antmux.**

Cette règle doit être vérifiée avant chaque installation ou modification de l’environnement.
